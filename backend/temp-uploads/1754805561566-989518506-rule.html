Rules of ID and Class in CSS

  • ID selector is defined with a # followed by the unique ID name.
  • Class selector is defined with a . followed by the class name.
  • ID must be unique within a page, while class can be used on multiple elements.
  • ID has higher specificity than class in CSS.
  • Use ID for single, unique elements; use class for reusable styles.
